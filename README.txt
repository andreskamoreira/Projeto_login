Projeto Login

O objetivo desse projeto é criar uma pagina de cadastro funcional e didática para possíveis 
clientes se cadastrem no site com suas informações pessoais.

Esse projeto foi criado utilizando HTML, CSS e Java Script, background com imagens, cores,
formulários, responsividades na parte de CSS e no Java Script para preenchimento e arquivamento
das informações do formulário, bem como validações, com visualização apenas no console.

Após o git clone, npm install e npm start.

Projeto ainda em conclusão.
